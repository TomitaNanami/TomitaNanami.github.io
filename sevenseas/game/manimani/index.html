mani mani

App game / LP design / Promotion video

Unity / Illustrator / Photoshop / Sketch / After Effects

mani maniは古事記のアマテラス神話の物語を飛び出す絵本にした
子ども向けiPadアプリです。
ジャイロ操作で周りを見渡しながら神様を探します。
神様をタッチすると、それぞれのアクションを起こします。
引きこもってしまったアマテラスを呼び戻すことができたらクリアです。
Unityで実装まで行いました。
うんちをばらまく神様がいるなど、子ども達が好きそうな要素が
たくさんありながらもなかなか知る機会のない古事記を
もっと知ってほしいと思い、このアプリを作成しました。
実際に子ども達にプレイしてもらったところ、
うんちを撒き散らすシーンでは大盛り上がりでした。

制作期間 App 5週間 / ­LPデザイン 1日 / プロモーションビデオ 1日
制作時期 2017年8~9月
